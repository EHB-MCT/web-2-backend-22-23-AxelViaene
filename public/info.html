API Documentation

GET / = root page (this page)

    USERS

  • GET /users = get all users
  • GET /users/:userid = get one user
  • POST /users/register = register user

    WEAPONS

  • GET /weapons = get all weapons

    MONSTERS

  • GET /monsters = get all monsters
  • GET /users/:monsterid = get one monster
  • POST /users/post = save one monster

    LINKED TABLES

  • GET /hunts = get all hunts
  • GET /user_greatswords = get all user_greatswords
  • DELETE /user_greatswords/delete = delete one user_greatswords
